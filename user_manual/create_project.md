# 新增Project

## 用途
為你的Project創建資訊

## 操作步驟
1. 點擊Project頁面右上角的 Create Project 按鈕
    
    ![Image1](images/create_project/project1.png)
        
2. 輸入Project名稱及描述後按下Create按鈕
        
    ![Image2](images/create_project/project2.png)
        
3. Project新增成功
        
    ![Image3](images/create_project/project3.png)
